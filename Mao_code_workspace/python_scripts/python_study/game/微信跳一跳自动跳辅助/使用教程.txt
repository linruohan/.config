
微信跳一跳自动跳辅助使用方法教程：http://www.smzy.com/smzy/down360517.html
1、在本站下载微信跳一跳自动跳辅助

2、然后，我们需要解压 zip 文档，打开文件夹：


3、打开系统自带的“终端”，用 cmd 就可以；

4、这是，我们需要通过终端进入文件夹，命令行如下：
～/Decktop/wechat_jump_game-master

5、安装 pip：在终端输入 sudo easy_install pip 再回车，可能要输入密码；

6、然后，在终端输入 pip install -r requirements.txt 再回车，系统会自动安装；requirements.txt 就是文件夹里的一个 txt 文档，里面写着会自动安装哪些程序；pip 就是第 5 步安装的程序，如果没安装，pip install -r requirements.txt 将无法执行；

7、接着继续安装 adb：打开下面的链接查看，有 3 种方法，建议用第二种，是英文，如果你不懂英文可以百度中文教程；

8、打开安卓手机的设置 - 开发者选项 - USB调试（如果没有开发者选项，可百度打开开发者选项的方法），用 USB 线连接手机和电脑，手机可能会弹出对话框，点同意；

9、在终端输入 adb devices，如果看到下面这种信息，说明 adb 已正确安装，也说明电脑成功检测到手机；
